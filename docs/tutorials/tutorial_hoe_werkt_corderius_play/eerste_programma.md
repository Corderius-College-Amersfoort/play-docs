# Eerste corderius-play programma

We gaan ons eerste programma schrijven in corderius-play

```python
import play 

play.start_program()
```

Kopieer de code naar je editor en draai de code.

Bij thonny betekent dit dat je op de groene play button klikt (Run current script).

Als het goed is, opent zich een leeg scherm op je computer van Python Play.
