# 1.1 Je eerste programma

Laten we eens kijken of het installeren gelukt is :).
Je kunt onderstaande video nadoen (Bekijk op YouTube voor volledig scherm)

De code in de video is:

```python
import play

play.start_program()
```

Als het goed is, opent er een scherm op je computer als je het programma uitvoert.
**play.start_program()** zorgt er dus voor dat een scherm geopend wordt.
Je mag op het kruisje klikken om dit af te sluiten.
