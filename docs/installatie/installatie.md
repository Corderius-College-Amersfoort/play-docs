# Installatie
Het is belangrijk dat je drie zaken installeert op je computer:
1. een programma waarin je code kunt typen en uitvoeren (een editor) 
2. de programmeertaal Python
3. het programma coderius-play

Er zijn veel opties. Hieronder geven we twee suggesties. Kies er zelf één of vraag je docent om advies.

## Optie 1: Thonny
Wil je gewoon snel aan de slag? 
Kies dan voor Thonny. Het is een simpele editor en we leggen je uit hoe je alles installeert [in deze tutorial](Thonny.md).

## Optie 2: Visual Studio Code
Wil je graag een editor die er mooier uitziet en ook veel meer kan?
Kies dan voor VS Code! Er is alleen een probleem: het is moeilijker om alles te installeren.
Vind je dat prima? Ga dan [naar onze tutorial](VScode.md)
